Word Wide Web General

What are the most essential technologies that made the World Wide Web such a success?

There are three main ingredients that make up the world wide web:

1. URL (uniform resource locator), which is the addressing scheme to find a document

2.HTTP (hypertext transfer protocol), which connects computers together

3.HTML (hypertext markup language), which formats pages containing hypertext links.

How are resources (html pages, images, etc) addressed in the web?

Html pages and images are adressed as as URL in the web or as a web address.

URLs consist of multiple parts including a protocol and domain namethat tell a web browser how and where to retrieve a resource.

What are the general components of a url?

A URL has two main compontents:

Protocoll indentifier for the URL. For example: http://example.com/

And a resource name ("example" in this case).

(for example HTML docs images and CSS stylesheets and JavaScript files)

What does a Web Server?

A web server connects to the internet and supports physical data intechange with other devices connected to the web.

How can content be stored on a web server?

Content stored on a server is divided into packets for transmission and is sent through special computers called outers that determine the best path for the content to travel through a series of wired and wireless networks to reach an internet service provider and ultimately a computer.

DOCTYPE tag is used to inform the browser about the version of HTML used in the document
